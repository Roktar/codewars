> 문제
  - 이 문제는 n*50의 값을 반환하는데, 이전 n값의 결과를 모두 더한 값을 반환한다.
  - n=1인 경우 50, n=2인 경우 150을 반환한다.( (2 * 50) + (1 * 50) = 150)
