> 문제
  - 이 문제는 주어진 괄호 문자열의 짝이 맞는지 아닌지를 찾아 true, false로 반환한다.

> 풀이 과정 및 결과
<pre>
   여는 괄호가 나올 때는 1씩 더하고 닫는 괄호가 나올 때는 1씩 빼서 결과를 구한다.
   도중에 마이너스값이 나오거나 최종적으로 0이 아닐 때는 짝이 맞지 않는 것으로 false를, 0일 때는 true를 반환한다.
</pre>
