# Pyramid Array - 20. 12. 03

이 문제는 입력값을 출력값의 형태로 만들어 반환한다.

- 기본 함수 (1)
  - pyramid(n: 정수)
    - 결과값 출력
- 기본적으로 주어지는 객체(0)
- 문제에서 요구하는 다른 함수(0)
- 추가함수(0)
- 추가클래스(0)
  - 구성함수(0)
- 추가객체(0)
- 추가변수(0)

- 입력 or 호출
  <pre>pyramid(3)</pre>
 
- 출력
  <pre>[[1], [1, 1], [1, 1, 1]]</pre>

> 문제
  - 이 문제는 n이 0일 때는 빈 배열, 1 이상일 때는 1, 2, ...n개 만큼의 길이를 갖는 서브 배열을 총 n개 갖는 2차원 배열을 만들어 반환한다.

> 풀이 과정 및 결과
<pre>
  n의 크기를 갖는 배열을 먼저 선언한 후 이 배열을 돌면서 순차적으로 크기가 1씩 증가하는 서브배열을 만든다.
</pre>
