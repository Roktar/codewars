# Bit Counting- 20. 07. 02

이 문제는 입력값을 출력값의 형태로 만들어 반환한다.

- 기본 함수 (1)
  - countBits(n: 정수)
    - 결과값 출력
- 기본적으로 주어지는 객체(0)
- 문제에서 요구하는 다른 함수(0)
- 추가함수(0)
- 추가클래스(0)
  - 구성함수(0)
- 추가객체(0)
- 추가변수(0)

- 입력 or 호출
  <pre>countBits(10)</pre>
 
- 출력
  <pre>2</pre>

> 문제
  - 이 문제는 주어진 정수의 2진수에서 1이 몇 개 있는지 그 수를 반환한다.

> 풀이 과정 및 결과
<pre>
toString(2)를 사용하여 2진화한 후 배열로 만들어서 1인 항목들만 필터링한 후 그 길이를 반환한다.
</pre>
