> 문제
  - 이 문제는 주어진 입력값에서 연속된 문자의 길이들과 그 문자를 찾는다.
  - 연속된 문자는 같은 문자로 연속된 경우를 말하며, 중간에 다른 문자가 나와서 끊겼다가 다시 이전에 나왔던 문자가 나와도 기존 것이 아닌 새로운 것으로 카운트한다.
    - ex) aaabbaa => [3, aaa], [2, bb], [2, aa]
